About this Website

This website is a simulation of the different processes that comprise an election. Starting from the constituencies and ending at the results, the site makes you travel trough the steps for an election to happen.

Given the numerous discrepancies in conducting an election in modern days, a simple website like this will handle all the complexities and show only what is required.


PS. This is only a simulation and cannot be linked to any official elections held anywhere or anytime.
